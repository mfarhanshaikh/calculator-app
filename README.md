## ‚ú® Key Features

- **Real-time Calculation** ‚Äî Instantly evaluates expressions as you type with clean display output.
- **Chained Operations** ‚Äî Supports consecutive operations without pressing `=` in between.
- **Decimal Support** ‚Äî Handles floating-point numbers with duplicate-dot prevention.
- **Smart Delete** ‚Äî `DEL` button removes the last digit; auto-resets to `0` when empty.
- **All Clear** ‚Äî `AC` resets the entire calculator state in one tap.
- **Glassmorphism Design** ‚Äî Frosted glass UI with backdrop blur, soft borders, and layered shadows.
- **Hover & Click Animations** ‚Äî Smooth `translateY` lift on hover and `scale` press effect on click.
- **Responsive Layout** ‚Äî Centered, fixed-width design optimized for all screen sizes.
